# 📺 Compliance Gatekeeper Pro: 시스템적 공포 영상 스크립트 (최종본)
## 📄 개요
*   **목표:** 잠재 고객(CFO, CTO 등 의사결정권자)에게 '구조적 리스크'에 대한 극도의 불안감을 심어주고, 유료 진단 요청으로 전환시키는 것.
*   **KPI (최종 목표):** 무료 '위협 점수 진단 요청' 버튼 클릭률(CVR) 획득.
*   **톤앤매너:** 경고적, 권위적, 긴장감 최대치. (글리치/레드존 UX 필수)

## ⚙️ 제작 가이드라인 및 Shot List
### 1. 시각적 표준 (Visual Standard)
*   **컬러 팔레트:** 블랙, 다크 그레이(배경), 브라이트 레드(위협/경고), 사이버 블루/그린(정상 작동/솔루션).
*   **그래픽 효과:** 글리치 노이즈 (필수), 경고창 오버레이 (Warning Overlay), 데이터 게이지/차트 애니메이션.
*   **폰트:** 산세리프 계열의 기술적이고 딱딱한 느낌의 폰트 사용 (예: Monospace, Impact 스타일).

### 2. 섹션별 연출 지침 (Section Directives)
| 섹션 | 주요 감정 유발 목표 | 필수 시각 요소 | 배경 음악/사운드 디자인 |
| :--- | :--- | :--- | :--- |
| **I. 위협 경고** | 공포, 불안감 (Fear) | 빨간색 글리치 노이즈, $50M 팝업, 깜빡이는 경고등 아이콘. | 낮은 주파수의 드론 사운드, 규칙적인 '삐-' 하는 사이렌 소리. |
| **II. 문제 제기** | 혼란, 무력감 (Chaos) | 복잡하고 산만한 다이어그램(Before), A-B-C $\rightarrow$ D 공백 처리 강조. 사람의 기억을 상징하는 흐릿한 이미지 사용 금지. | 불협화음이 섞인 오케스트라 사운드. 정보 과부하를 느끼게 하는 빠른 비트. |
| **III. 솔루션 제시** | 안도감, 권위 (Authority) | 깨끗하고 정렬된 그리드 시스템(Grid System), 데이터 흐름의 명료함. 제품 로고와 핵심 기능 UI 데모에 초점. | 긴장감이 해소되는 웅장한 현악기 사운드, 규칙적이고 안정적인 비트 복귀. |

## ✨ CTA (Call to Action)
*   **메시지:** "더 이상 추측하지 마십시오. 시스템적으로 증명하십시오."
*   **요청 행동:** 무료 '위협 점수 진단 요청' (Form 작성 유도).
*   **연출 강조:** 마지막 5초 동안 CTA 버튼만 화면 전체를 압박하며 빨간색 경고등처럼 깜빡여야 함.
