# 🛠️ Funnel Conversion Test Checklist (Priority: High)**목표:** $49 진단 보고서 구매까지의 고객 여정(User Journey)에서 메시징과 UI가 논리적 흐름을 따르는지 검증한다.**테스트 범위:** MVP $\rightarrow$ Paywall $\rightarrow$ Checkout Flow**섹션 1. 리스크 노출 (P2)**- [ ] **[필수] Total Risk Exposure 값 계산 테스트:** 시스템이 가짜 데이터(Mock Data)를 입력받았을 때, 연간 사업 규모의 X% 수준으로 위험 값을 정확히 산출하여 화면에 표시하는지 확인한다. (정량적 공포감 유발).**- [ ] **[필수] Red Zone 경고 활성화:** 리스크 값이 특정 임계점(Threshold)을 넘으면, 시각적으로 가장 강력한 'Red Zone' 효과와 함께 비상 메시지가 뜨는지 검증한다.**섹션 2. Paywall 인터랙션 (P3)**- [ ] **[필수] $49 포지셔닝 테스트:** 가격이 단순 서비스 비용이 아닌, '최소 필수 진단/조사 비용(Investigation Fee)'으로 정의되어 명시적으로 설명되는지 확인한다.**- [ ] **[중요] Value Gap 비교 UI:** "현재 (불확실) vs. $49 보고서 구매 시 (명확)"의 대비 구조를 통해 가치 격차를 극대화하는 카피가 노출되는지 검증한다.**섹션 3. 전환 메시징 및 유효성 (P4)**- [ ] **[필수] Urgency Trigger 테스트:** 결제 버튼 근처에 '지금 행동하지 않으면 발생하는 기회비용'을 상기시키는 문구(TDR 관련)가 노출되는지 확인한다.**- [ ] **[중요] 다음 단계 유도 (Lock-in):** $49 구매 완료 페이지에서 Silver Tier($1,999/월)로의 자연스러운 업셀링 경로와 할인이 안내되는지 검증한다.
