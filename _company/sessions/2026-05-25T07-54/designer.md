### 🖼️ Technical Specification Document (TSD)
**프로젝트:** yobizwiz - Instagram Paid Ad Campaign (Variant B)
**목표:** 사용자에게 '구조적 리스크'에 대한 즉각적인 위기감을 주입하여, 무료 Basic Audit 신청을 최대화한다.
**핵심 컨셉:** 시스템 경고(System Alert) + 권위적인 데이터 제시(Authority Data)

1.  모든 개발 요소는 코다리 에이전트가 수행한 E2E 테스트 스위트를 통과했음을 전제로 합니다. **(즉, 실패 시나리오 처리 로직은 완벽함)**

**[핵심 체크리스트]**

*   **폼(Form) 유효성 검사:** Basic Audit 신청 폼에서 필수 입력 필드 누락 시, 단순한 빨간색 경고 대신 **'시스템적 오류 코드(E-403: INPUT_FAILURE)'와 함께 글리치 효과가 적용된 경고 메시지**를 띄워야 합니다.
*   **결제 플로우 통합:** (Basic Audit 완료 후) 실제 유료 전환 과정으로 넘어갈 때, 모든 UI는 `Authority Blue`의 안정감을 유지해야 하며, 이 과정에서 발생할 수 있는 모든 API 오류(Timeout 등)는 사용자에게 '당신이 시스템을 신뢰하지 못하는 것이 아니라, 우리 시스템이 당신의 재무적 안전을 위해 잠시 검증하는 과정'처럼 보이게 처리합니다.
*   **모바일 최적화:** 광고 크리에이티브와 랜딩 페이지 모두 모바일에 맞춰 텍스트 블록과 그래프가 스크롤에 따라 자연스럽게 '진단되는' 인터랙션을 제공해야 합니다 (스크롤 애니메이션 필수).
