## 📲 인스타그램 릴스 카피 세트 (총 3가지)
*(권장 길이: 15~20초. 화면 전환 속도 빠르고, 배경 음악은 긴박하고 고권위적인 느낌으로 설정)*

### 🔥 Set 1: AI 과신 리스크 - '환각'에 대한 공포 자극
**[주제]** LLM 사용의 편리함 뒤에 숨겨진 법적 책임 전가 위험 (AI Hallucination Liability)
**[타깃 감정]** 오만함 → 패닉

| 요소 | 시간대 (초) | 스크립트 텍스트 (나레이션/화면 자막) | 비주얼 지시 및 효과 |
| :--- | :--- | :--- | :--- |
| **HOOK (0~3초)** | 0-3s | **"AI가 만든 보고서? 이걸로 법적 공방에 나설 수 있습니까?"** (강력한 질문형) | 배경: 빠르고 자신감 넘치는 AI 화면 → 갑자기 빨간 글리치 노이즈와 함께 정지. 자막: 🚨 위험 경고! |
| **문제 제기 (4~10초)** | 3-7s | "LLM은 최고의 도구지만, 법적 판단은 '예측 아키텍처'가 필요합니다. AI의 환각 정보는 당신 회사의 무효화 사유가 될 수 있습니다." | 화면 전환: 깔끔한 문서 → 깨진/왜곡된 데이터 구조 이미지. (신뢰도 하락 시각화) |
| **핵심 메시지 (11~15초)** | 7-12s | "단순히 규제를 검색하는 건 '검토'일 뿐입니다. 법규 준수를 시스템적으로 *강제*해야 합니다." | 자막 강조: 단순 검토 $\rightarrow$ 구조적 강제(Compliance Gateway) |
| **CTA (16~20초)** | 15-20s | "당신의 비즈니스 프로세스에 '법적 구멍'이 없는지, 지금 무료로 진단받으세요. [프로필 링크] 클릭." | 화면 전체를 빨간색/경고 색상으로 가득 채우며 CTA 문구 강조. (강한 시각적 압박) |

### 💧 Set 2: 데이터 유출 리스크 - '익숙함'에 대한 경계심 자극
**[주제]** PII 마스킹 실패 및 클라우드 전송의 치명적인 법규 사각지대 (PII Masking Failure)
**[타깃 감정]** 안도감 → 공포

| 요소 | 시간대 (초) | 스크립트 텍스트 (나레이션/화면 자막) | 비주얼 지시 및 효과 |
| :--- | :--- | :--- | :--- |
| **HOOK (0~3초)** | 0-3s | **"데이터를 클라우드에 올리는 순간, 당신의 회사는 이미 위험합니다."** (가장 일상적인 행위를 위협) | 배경: 평화로운 사무실/클라우드 전송 화면 → 갑자기 경고음과 함께 데이터 조각들이 흩어지는 연출. 자막: ⚠️ 지금 당장 확인하세요! |
| **문제 제기 (4~10초)** | 3-7s | "PII 마스킹 실패는 단순한 보안 문제가 아닙니다. 수억 원대 벌금과 집단 소송을 부르는 '구조적 리스크'입니다." | 화면 전환: 완벽해 보이는 데이터베이스 → 특정 필드(이름, 주민번호 등)가 노출되는 클로즈업 샷. (충격적인 근접 촬영) |
| **핵심 메시지 (11~15초)** | 7-12s | "혹시 '이 정도면 괜찮겠지'라고 안심하셨습니까? 법적 리스크는 바로 그 사각지대에서 터집니다." | 자막 강조: 안일한 판단(❌) $\rightarrow$ 시스템 점검 필수(✅). (시스템적 무지 지적) |
| **CTA (16~20초)** | 15-20s | "당신 회사의 데이터 흐름을 법적 관점에서 재설계하세요. [프로필 링크]에서 무료 리스크 진단 체험 신청." | 전문적인 보고서 이미지와 함께 CTA 문구 제시. 신뢰감과 긴급성을 동시에 전달. |
