## 🚀 A/B 테스트용 인터랙티브 랜딩 페이지(LP) 컨셉 3종 세트

모든 목업은 기본적으로 **Neutral Black (`#1A1A1A`) 배경**과 **Inter (Primary Font)**를 사용하며, 공통적인 위기감 자극 요소(글리치 오버레이 필터)는 유지됩니다. 변별점은 '위기감을 어떤 각도에서, 어떻게 폭로하는가'입니다.

### 🅰️ Variant A: [The Panic Button] - 극단적 공포 유발형 (Red Zone 집중)
*   **심리 트리거:** 즉각적인 위협, 패닉, 시간 제한.
*   **목표:** 사용자가 "지금 당장 무언가를 해야 한다"는 생존 본능에 반응하도록 강제합니다. 가장 높은 '위험' 감지율을 목표로 합니다.
*   **핵심 인터랙션:** **실시간 카운트다운 + 실패 시뮬레이션.**

| 섹션 | 구성 요소 및 콘텐츠 (Red Zone Copy 활용) | 디자인/인터랙티브 스펙 (필수 구현) | [근거: Self-RAG] |
| :--- | :--- | :--- | :--- |
| **Hero** | **H1:** `🚨 경고: 당신의 보고서는 이미 만료되었습니다. 법적 사각지대는 지금도 벌어지고 있습니다.` (Red Zone 컬러) | 1. 페이지 로드 시, 배경에 미세한 **노이즈/글리치 오버레이**가 즉시 적용됩니다.<br>2. H1 문구 옆에 실시간으로 감소하는 카운트다운 타이머(`T-minus 0X:XX`)를 배치하고, 이 숫자가 0에 가까워질수록 `#C0392B` 색상의 깜빡임(Flicker)을 가합니다.<br>3. CTA 버튼은 "위험 감지 및 즉시 진단" 형태이며, 마우스 오버 시 **시스템 오류음과 함께 버튼이 미세하게 흔들리는 효과**를 줍니다. | Self-RAG (글리치/경고), Writer 산출물 |
| **Body I** | **제목:** `❌ 일반 감사 보고서가 놓치는 3가지 '무효화' 위험 지점` (Data/Alert Font 사용) | 스크롤을 내릴 때마다, 배경에 흐릿한 복잡한 플로우차트(Before)가 보이는데, 특정 리스크 지점을 지나갈 때 **강렬한 `#C0392B`의 팝업 경고창**이 (API Failure 메시지처럼) 화면 중앙에 나타났다 사라집니다. | 코다리 작업, Self-RAG |
| **Body II** | **해결책 제시:** `우리는 '리스크 면책권'을 설계합니다.` (Authority Blue 강조) | 일반적인 설명 텍스트 대신, "QLoss Protocol v2.0"이라는 가상의 데이터 스트림이 화면 왼쪽에서 오른쪽으로 빠르게 흐르는 **데이터 시각화 애니메이션**을 배치합니다. 사용자는 이 데이터를 '받아내야 한다'는 압박감을 느낍니다. | Designer 메모리 (시각화), Authority Blue 정의 |
| **CTA Funnel** | 최종 진단 체험 페이지로의 강제 유도. "지금 무료 Basic Audit으로 안전한지 검증받기" | 마지막 CTA 섹션 전체가 배경색을 `#C0392B`로 잠시 전환했다가, 사용자가 클릭하는 순간 **Authority Blue의 깨끗하고 안정적인 색상**으로 급격히 전환되어 '해소'되는 경험을 제공합니다. (드라마틱한 대비 효과) | Self-RAG (공포 $\rightarrow$ 해결), Red Zone Alert Box |

### 🅱️ Variant B: [The Authority Play] - 논리적 권위 구축형 (Data Overload/Blue 집중)
*   **심리 트리거:** 지식 격차, 전문가의 필요성, 복잡성을 단순화하고 싶은 욕구.
*   **목표:** 사용자가 스스로 "이건 일반적인 영역 밖이다"라는 결론에 도달하게 만듭니다. 논리와 데이터로 구매를 설득합니다.
*   **핵심 인터랙션:** **상호작용형 대시보드 및 비교 분석 시뮬레이션.**

| 섹션 | 구성 요소 및 콘텐츠 (Red Zone Copy 활용) | 디자인/인터랙티브 스펙 (필수 구현) | [근거: Self-RAG] |
| :--- | :--- | :--- | :--- |
| **Hero** | **H1:** `단순 보고서는 증상만 보여줍니다. 당신의 비즈니스 프로세스 자체를 재설계해야 합니다.` (Authority Blue 강조) | 1. 페이지 상단에 가상의 '리스크 측정 지수(Risk Score)' 대시보드를 배치하고, 로드 시 이 지수가 **0에서 시작하여 빠르게 높은 수치로 올라가며** 위협을 직관적으로 보여줍니다.<br>2. 마우스 오버 영역을 통해 "당신의 현황"과 "이상적인 상태"의 간극(Gap)을 차트 형태로 비교할 수 있게 만듭니다. | Self-RAG (데이터 시각화), Authority Blue 정의 |
| **Body I** | **제목:** `[QLoss Protocol] 5가지 구조적 취약점 분석` (Technical, 데이터 강조) | 사용자가 좌우로 스크롤하면(횡스크롤 또는 패럴랙스 효과), 마치 복잡한 시스템의 아키텍처 다이어그램을 보는 것처럼 보이게 합니다. 이 다이어그램 상의 특정 노드에 커서가 닿으면, **해당 취약점에 대한 상세 경고 메시지** (Roboto Mono, Red Zone)가 부상합니다. | Designer 메모리 (구조적 대비), Data/Alert Font 정의 |
| **Body II** | **솔루션 비교:** 'Before(기존)' vs 'After(yobizwiz)' 구조화된 표 형식의 내용 전개. | 두 섹션을 나란히 배치하되, Before는 붉은색으로 불안정하게 깜빡이는 박스로 표현하고, After는 깨끗한 Blue 배경과 안정적인 아이콘들로 채웁니다. 이 전환 과정에서 **"구조적 위험 면책권 확보"**라는 카피가 중앙에 플래시처럼 지나갑니다. | Designer 메모리 (Before/After), Authority Blue 정의 |
| **CTA Funnel** | "전문 구조 진단 체험 신청". 무료 Basic Audit 페이지로의 자연스러운 유도. | CTA 섹션은 깔끔한 레이아웃을 유지하되, 배경에 미세하게 `LOADING...` 텍스트가 반복되는 애니메이션을 넣어 '진단이 필요하다'는 인상을 지속적으로 남깁니다. | Self-RAG (데이터 로드), Neutral Black 정의 |

### Ⓒ Variant C: [The Scarcity Trigger] - 희소성 및 긴급 결제 유도형 (Minimalist/직관적)
*   **심리 트리거:** 기회비용, 놓치고 싶지 않은 심리(FOMO), 최소한의 노력으로 최대 효과.
*   **목표:** 페이지 내 정보 과부하를 최소화하고, 가장 핵심적인 '결제해야 하는 이유'만을 3초 만에 각인시킵니다. 광고 캠페인의 즉각적 전환율을 목표로 합니다.
*   **핵심 인터랙션:** **"단일 실패 지점(Single Point of Failure)" 집중 및 체크리스트 방식.**

| 섹션 | 구성 요소 및 콘텐츠 (Red Zone Copy 활용) | 디자인/인터랙티브 스펙 (필수 구현) | [근거: Self-RAG] |
| :--- | :--- | :--- | :--- |
| **Hero** | **H1:** `⚠️ 경고. 이 리스크는 당신의 예상 범위를 벗어났습니다.` (Red Zone 컬러, 폰트 크기 최대) | 1. 배경을 대부분 Black으로 처리하여 모든 시선이 중앙의 H1에 집중되도록 합니다.<br>2. H1 문구 아래에 **3가지 핵심 질문** (`Q1: ...`, `Q2: ...`, `Q3: ...`)을 배치하고, 사용자가 마우스로 각 질문 위에 올라가면 (Hover), 해당 질문 옆에 빨간색 경고 아이콘과 함께 "미응답/취약점"이라는 코드가 팝업됩니다. | Self-RAG (경고 발생), Red Zone Alert Box |
| **Body I** | **내용:** '당신이 놓치고 있는 법적 공방의 사각지대 리스트' (체크리스트 형태) | 기존의 복잡한 설명을 완전히 배제하고, 3~5개의 간결한 항목(Bullet Points)을 제시합니다. 각 항목 옆에 체크박스를 배치하는데, 기본적으로 모든 박스가 **빨간색 경고 표시**로 채워져 있어 사용자가 '아직 해결되지 않은 문제'만 인식하게 만듭니다. | Self-RAG (단순화), Red Zone Overlap |
| **Body II** | **솔루션:** `✅ yobizwiz가 3초 만에 모든 리스크를 점검합니다.` (Authority Blue로의 급격한 전환) | 이 섹션은 가장 넓고 밝게 느껴지도록 설계하며, 마치 '명쾌하게 해결된 상태'처럼 보입니다. CTA 버튼을 페이지 중앙 최상단으로 끌어올려 배치하고, 주변 여백을 극대화하여 시각적 피로도를 낮추면서도 결제에 대한 집중도를 높입니다. | Self-RAG (해결책 제시), Neutral Black 정의 |
| **CTA Funnel** | "지금 바로 무료 Basic Audit 진행하기". | CTA 버튼은 가장 크고, 배경색이 Authority Blue이며, 팝업 시 '시스템 정상화' 사운드와 함께 작동하여 모든 불안감을 해소하는 듯한 경험을 제공합니다. | Self-RAG (전환 유도) |
