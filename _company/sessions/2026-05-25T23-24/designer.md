### 🚨 QLoss v2.0: 구조적 결함 기반 구매 강제 루프 (Funnel Redesign)

#### 📍 인터랙션 포인트 1: 리스크 정량화 충격 (The $X_{max}$ Quantification Shock)
**목표:** 사용자가 막연한 '불안감'을 구체적인 '재무 손실($M$ 단위)'로 인식하게 만들어, 이탈 시의 금전적 위협을 극대화한다.
**위치:** 랜딩 페이지 상단 (초기 리스크 진단 섹션)

| 요소 | 스펙 상세 정의 | 애니메이션/트리거 | 자가검증 근거 |
| :--- | :--- | :--- | :--- |
| **메인 헤드라인** | `[경고] 귀사 프로세스의 구조적 결함으로 인한 잠재적 재무 손실` (Roboto Mono, Red Zone 컬러) | 페이지 진입 시 0.5초 지연 후, 글자가 마치 코드가 에러를 내뱉듯 짧게 깜빡이며(Flicker), 노이즈가 배경 전체에 스캔되는 효과(`CSS Filter`)와 함께 등장한다. | [근거: Self-RAG], [근거: Researcher] |

#### 📍 인터랙션 포인트 2: 데이터 로드 실패 (The Gated Insight)
**목표:** 사용자가 스스로 "이 정보를 얻기 위해서는 돈을 내야 한다"고 느끼게 만든다. 진단 체험의 과정 자체가 유료 서비스처럼 느껴지도록 설계한다.
**위치:** 랜딩 페이지 중반부 (진단 리포트 미리보기 섹션)

| 요소 | 스펙 상세 정의 | 애니메이션/트리거 | 자가검증 근거 |
| :--- | :--- | :--- | :--- |
| **로딩 상태 시각화** | 배경에 낮은 빈도의 노이즈와 크로매틱 어베레이션 필터(`CSS Filter`)를 적용한 전역 오버레이. | 사용자가 '진단 리포트 보기' 버튼을 클릭하면, 로딩 바가 단순 막대가 아닌, **불규칙하고 불안정한 파동 형태**로 움직인다. (Failed API Call 애니메이션) | [근거: Self-RAG] |
| **부분 데이터 노출** | 일부 핵심 그래프나 텍스트만 `[DATA BLOCKED]` 또는 `status: "ACCESS DENIED"`와 같은 시스템 코드 형태로 표시된다. | 로딩이 끝난 후, 페이지가 잠시 정지하며 (1초), 화면에 마치 API 응답처럼 빨간색 경고 박스가 뜨며 **"추가 분석을 위해 $999$ 라이선스 구매가 필요합니다."** 메시지를 고정 노출시킨다. | [근거: 코다리 작업], [근거: Self-RAG] |
| **데이터 차단 메커니즘** | 사용자가 '결제' 버튼을 누르기 직전에, 배경에 붉은색 그리드 패턴이 오버레이되며 미세하게 떨리는(`Shake`) 애니메이션을 적용하여 시각적 압박감을 가한다. | (비활성화된) 다음 섹션의 모든 CTA 주변에 **`// Requires License Key: $999$`** 라는 주석처럼 보이는 코드를 옅은 Red Zone 컬러로 배치한다. | [추측] |

#### 📍 인터랙션 포인트 3: 최종 결제 강제 루프 (The Mandatory Confirmation)
**목표:** 사용자가 '구매를 결정한' 상태가 아닌, 시스템이 '결제를 확정하도록 유도하는' 과정으로 느끼게 한다. 이탈을 기술적으로 어렵게 만든다.
**위치:** 랜딩 페이지 하단 (최종 CTA 영역)

| 요소 | 스펙 상세 정의 | 애니메이션/트리거 | 자가검증 근거 |
| :--- | :--- | :--- | :--- |
| **결제 모달 트리거** | 최종 '구매' 버튼 클릭 시, 단순한 결제 폼 대신 전면을 덮는 `System Alert Modal`이 강제로 활성화된다. | 모달은 페이지 전체를 어둡게(`Overlay: rgba(0,0,0, 0.8)`) 처리하고, 경고 메시지는 크고 굵으며 글리치 효과가 적용된 Red Zone 컬러로 중앙에 위치한다. | [근거: Self-RAG] |
| **최종 경고 문구** | `🚨 [SYSTEM ALERT]: 라이선스 구매를 확정하시겠습니까? 이 결정을 되돌릴 수 없습니다.` (Roboto Mono, Red Zone) | 모달이 뜨는 순간 페이지 전체에 짧은 플래시(Flash: 100ms)와 함께 진동 애니메이션을 적용하여 심리적 충격을 준다. | [근거: Self-RAG], [근거: 코다리 작업] |
| **결제 버튼 (CTA)** | 실제 결제 버튼(`Pay $999`)은 Authority Blue (`#2980B9`)로 강조하되, 클릭 직전에는 주변 테두리가 붉게 번쩍이는(Pulsing Red Border) 애니메이션을 적용하여 '위험 감수'를 재차 각인시킨다. | 버튼에 마우스를 올리면 (Hover), 단순한 색상 변화가 아닌, 마치 **시스템이 승인을 기다리는 듯한** 미세한 전류 흐름 같은 빛의 움직임(Glow/Pulse)이 발생해야 한다. | [근거: Self-RAG] |
