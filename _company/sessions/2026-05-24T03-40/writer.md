# 🚨 MANDATORY COPY KIT: 최종 배포 버전 (Tone & Manner Lock Down)
*본 카피는 모든 마케팅 접점(웹사이트, 광고, 영상 CTA 등)에서 일관되게 사용되어야 합니다. 어떠한 '부드러운' 표현도 허용되지 않습니다.*

## 1. [CTA 버튼 문구] (Call to Action - 행동 강제)
버튼을 누르는 행위 자체에 심리적 압박과 긴급성을 부여합니다.

| 사용 환경 | 기존(가정) | **최종 권고 카피** | 핵심 전략/효과 |
| :--- | :--- | :--- | :--- |
| 무료 진단 페이지 | 무료 리스크 점검 받기 | **[🚨 즉시] $QLoss$ 실시간 계산 및 감사 요청** | '무료'라는 단어 삭제. 손실액($QLoss$)과 '즉시', '감사' 등 고권위 용어로 대체하여 행동의 필요성을 극대화합니다. |
| 웹사이트 메인 CTA | 전문가와 상담하기 | **[필수] 컴플라이언스 게이트웨이 구축 설계 요청** | 단순한 문의가 아닌, 솔루션 도입 자체가 '필수적인 생존 과정'임을 명시합니다. |
| 데모 신청 페이지 | 서비스 체험해보기 | **시스템 권한 확보: 15분 리스크 진단 시뮬레이션 실행** | '체험'이라는 가벼운 단어를 배제하고, 시스템 접근권(Authority)을 얻는다는 느낌으로 격상시킵니다. |

## 2. [경고 및 공포 유발 카피] (Warning & Fear Trigger - 불안감 증폭)
사용자가 자신의 현재 상태를 의심하게 만듭니다. Red Zone 효과와 함께 사용되어야 합니다.

| 사용 환경 | 기존(가정) | **최종 권고 카피** | 핵심 전략/효과 |
| :--- | :--- | :--- | :--- |
| 메인 히어로 섹션 | 혹시 놓치고 있는 부분이 있을까요? | **경고: 현재의 프로세스는 법적 무효화 위험에 노출되어 있습니다.** | '혹시' 대신 강한 확정적 경고를 사용합니다. 리스크의 결과를 명확히 지목합니다. |
| 기능 소개 섹션 | 수동 검토로는 한계가 있습니다. | **[D-Gap Alert] 사후 점검만으로는 법적 방어벽이 될 수 없습니다.** | '한계' 대신 시스템적인 구멍(D-Gap)을 명시하여, 사용자가 자신의 프로세스에 결함이 있음을 스스로 인식하게 합니다. |
| 가격/제한 섹션 | 혜택이 곧 종료됩니다! | **[🚨 치명적 경고] 감사 라이선스 만료 임박: $QLoss$ 계산 기능 정지 예정.** | '할인'이나 '만료' 같은 상업적 언어 대신, 핵심 기능의 *상실*과 그로 인한 재정적 피해를 경고합니다. |

## 3. [제품 설명 및 가치 제안 카피] (Value Proposition - 해결책 제시)
솔루션이 단순한 도구가 아님을 강조하며 '권위'를 부여합니다.

| 사용 환경 | 기존(가정) | **최종 권고 카피** | 핵심 전략/효과 |
| :--- | :--- | :--- | :--- |
| 도입 문구 (Lead Copy) | 컴플라이언스 관리를 도와드립니다. | **정보 수집이 아닌, 법규 준수를 '구조적으로 강제'하는 예측 아키텍처가 필요합니다.** | 핵심 가치를 기능 나열이 아닌, 학술적/시스템 공학 용어(예측 아키텍처)로 정의하여 권위를 높입니다. |
| 차별점 강조 (USP) | 실시간으로 법규를 비교해요. | **[실시간 전 세계 DB 매칭] 데이터 입력과 동시에 리스크 경고가 발동됩니다.** | '비교' 대신 '매칭', '발동' 등 시스템 작동 용어를 사용하여 기계적, 절대적인 정확성을 강조합니다. |
| 마무리 문구 (Closing) | 지금 바로 시작하세요! | **더 이상 추측에 의존하지 마십시오. 법적 방어벽을 구축하십시오.** | 감성적 동기 부여를 제거하고, '의무'와 '책임감'이라는 이성적/법률적 압박감을 사용합니다. |
