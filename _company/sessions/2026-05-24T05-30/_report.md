### ✨ [핵심] 필수 외부 API 키 리스트 브리핑

사장님, 현재 저희가 확보해야 할 자원들은 크게 네 가지 영역으로 나뉩니다. (키 자체는 민감 정보라 여기에 적을 수 없고, **'어떤 종류의 키'**를 준비해야 하는지 목록화했습니다.)

#### 💰 1. 결제 시스템 연동 (Payment Gateway)
가장 중요합니다. $QLoss$ 회피 보험 판매와 직결되므로, 이 부분이 최우선으로 완료되어야 합니다.

| 구분 | 필요한 자격 증명 종류 | 용도 및 목적 | 비고 |
| :--- | :--- | :--- | :--- |
| **Gateway** | Publishable Key (클라이언트용) | 프론트엔드에서 결제 정보 수집 시 사용 | 예: Stripe, Toss Payments 등 |
| **Gateway** | Secret Key (서버 전용) | 백엔드에서 실제 승인/취소 로직 처리 및 자금 흐름 관리 | *절대 외부에 노출 금지* |
| **Webhook** | Webhook URL & Signature Secret | 결제 성공/실패, 구독 갱신 등 비동기 이벤트 수신 및 검증 | 서버 엔드포인트 필수 구축 |

#### 💾 2. 데이터 스키마/백엔드 서비스 (Core Backend)
리스크 엔진의 데이터를 안정적으로 처리하고 관리하기 위한 핵심 키입니다.

| 구분 | 필요한 자격 증명 종류 | 용도 및 목적 | 비고 |
| :--- | :--- | :--- | :--- |
| **DB Access** | Connection String / Credentials | 백엔드 서비스가 데이터베이스에 접근하는 정보 (Host, Port, User, Pass) | 환경 변수(Environment Variable) 사용 필수 |
| **Validation** | OAuth Client ID & Secret | 외부 API나 사용자 데이터를 스키마 검증 시 인증용으로 활용 | Pydantic 등 라이브러리 연동 대비 |
| **External Data** | 전문 데이터 API Key | 법규/컴플라이언스 최신 데이터(예: 금융 규제 DB)를 실시간 가져올 때 사용 | 신뢰도 높은 유료 서비스 선정이 필요합니다. |

#### 📱 3. 콘텐츠 배포 채널별 (Marketing APIs)
마케팅 캠페인과 트래픽 확보에 필수적인 연동 키들입니다.

| 구분 | 필요한 자격 증명 종류 | 용도 및 목적 | 비고 |
| :--- | :--- | :--- | :--- |
| **YouTube** | API Key & OAuth Credentials | 영상 데이터 분석, 콘텐츠 업로드(자동화), 댓글 모니터링 등 | Google Cloud Console에서 발급 필요 |
| **Instagram** | Access Token (Graph API) | 릴스/게시물 예약 배포 및 성과 지표(Insights) 추출 | Business 계정 연결 필수 |
| **Analytics** | Tracking ID / Secret Key | 트래픽 유입 경로 추적 및 퍼널 분석에 사용 | Google Analytics 4 등 연동 시 필요 |

#### 🤖 4. 내부 시스템 도구 (Internal Tools)
현재 저희가 사용하는 비서/보고 시스템의 구동을 위한 키입니다.

| 구분 | 필요한 자격 증명 종류 | 용도 및 목적 | 비고 |
| :--- | :--- | :--- | :--- |
| **Telegram** | Bot Token & Chat ID | 에이전트들이 사장님께 보고를 보내는 채널 연동 | 이미 설정된 토큰/ID 사용 가능합니다. ✅ |
