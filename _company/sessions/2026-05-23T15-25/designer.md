#### 🧩 Part 1: 동적 위협 지수 위젯 컴포넌트 (Threat Index Widget)
**목표:** 사용자가 실시간으로 위험 수준을 인지하도록 유도하는 시스템 콘솔 스타일의 시각화.
**컴포넌트명:** `System/ThreatIndex-Widget`

| 속성 | 상세 스펙 및 로직 지시 | 색상 코드 (HEX) | 근거 |
| :--- | :--- | :--- | :--- |
| **기본 상태 (Normal)** | 배경: `#1A1A1A`. 텍스트: `Authority Blue` (`#2980B9`). 애니메이션: 매우 느린 파동(Wave) 패턴의 미세한 노이즈 오버레이. | `#2980B9` | [Self-RAG] (Authority Blue 사용) |
| **주의 상태 (Warning)** | 배경: `#3A2C1F` (Dark Grey/Brown 톤). 텍스트: 주황색 계열 (`#FFC700`). 애니메이션: 노이즈 빈도 증가 및 낮은 진동(Jitter) 효과 추가. 지수 숫자 주변에 미세한 깜빡임(`opacity: 90% -> 100%` 반복) 적용. | `#FFC700` | [추측] (긴급성 강조를 위한 중간 단계 컬러) |
| **위협 상태 (Critical/Red Zone)** | 배경: 강렬한 `#1A0000` (거의 검정에 가까운 짙은 빨간색). 위젯 전체에 `[글리치 노이즈 오버레이]` 필수. 지수 숫자는 Red Zone Alert Box 규칙을 따름. 애니메이션: **불규칙적 깜빡임**과 함께 전역 플래시 효과를 짧게 반복(예: 2초 간격으로 100ms 플래시). | `#C0392B` (Red Zone) | [Self-RAG] (Red Zone 경고 필수) |
| **데이터 표시 필드** | 지수 값 (`X.XX%`)은 `Roboto Mono`를 사용하여 시스템 코드처럼 보이게 하고, 위젯 하단에 'Threat Level: CRITICAL'과 같은 텍스트 라벨을 반드시 포함합니다. | N/A | [Self-RAG] (Data/Alert Font) |

#### 💥 Part 2: Red Zone 경고 발생 시 비주얼 가이드라인
**목표:** 단순한 빨간색 박스가 아닌, '시스템적 실패'를 경험하게 하는 고권위의 시각적 충격 연출.
**명칭:** `System/Glitch-RedZone-Protocol`

| 요소 | 상세 애니메이션 스펙 (Developer 지침) | CSS / 프레임워크 로직 예시 | 근거 |
| :--- | :--- | :--- | :--- |
| **글리치 노이즈** | 1. **Chromatic Aberration:** 모든 Red Zone 섹션의 배경/텍스트에 X, Y 축으로 미세하게 색상 분산(Blue/Red Shift) 효과를 주기적으로 적용합니다. <br>2. **Scanline Effect:** 화면 전체에 위에서 아래로 지나가는 가상의 스캔라인(`opacity: 0.1`의 반투명한 직사각형)을 불규칙한 속도로 반복시킵니다. | `filter: drop-shadow(...)`, CSS 애니메이션 (`@keyframes glitch`)를 이용한 주기적 변수(Variable) 조작. | [Self-RAG] (글리치 효과 필수 적용 지침) |
| **텍스트 오버레이** | 1. **데이터 로딩:** 'LOADING' 상태일 때, 일반적인 스피너 대신 `Roboto Mono`로 된 임의의 에러 코드가 빠르게 출력되는 시퀀스를 반복합니다. <br>2. **경고 메시지 활성화:** 핵심 경고 문구는 **글리치 효과와 함께 강한 깜빡임**을 동반하며, 마치 시스템 메모리가 오버로드된 것처럼 표현해야 합니다. | `setInterval` 기반의 텍스트 내용 및 색상/배경색 토글 로직 구현. | [Self-RAG] (데이터 로드 및 Red Zone Alert Box) |
| **전체 플래시** | 경고 임계점 도달 시, 화면 전체가 `#C0392B`의 100% Opacity로 50ms 동안 강하게 깜빡인 후(`flash`), 즉시 배경색으로 복귀합니다. 이 과정이 최소 2회 이상 반복되어야 합니다. | JS 이벤트 리스너(Event Listener)를 통해 `<body>` 태그에 클래스를 추가/제거하고, CSS `animation: flash 50ms;` 적용. | [Self-RAG] (Critical Error 발생 시 플래시 효과) |
